Banco de Dados de Criaturas Paranormais

O Sistema de Registro de Criaturas é o banco de dados fundamental da Ordem, documentando detalhes, comportamentos e métodos de contenção para entidades paranormais. Após destruição parcial, sua reconstrução é urgente e vital, pois cruza dados para prever riscos e transformar o desconhecido mortal em conhecimento que salva vidas.
